# Salmonella Serovar Wiki

A curated web portal providing comprehensive, serovar-specific information on *Salmonella* serovars for academia, industry, and public health.

**Live site:** https://fsl-mqip.github.io/salmonella-serovar-wiki/

Migrated from the [original Confluence Wiki](https://confluence.cornell.edu/display/FOODSAFETY/Salmonella+Serovar+Wiki).

## Local Development

```bash
pip install -r requirements.txt
mkdocs serve
```

Then visit `http://127.0.0.1:8000`.

## Deployment

The site is deployed to GitHub Pages automatically via GitHub Actions on push to `main`.
