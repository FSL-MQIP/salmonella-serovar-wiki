# *S.* Korovi

Salmonella Korovi

## Background Information

 *Salmonella enterica *subsp.* enterica* serovar Korovi (antigenic formula 38:g,m,[s]:=E2=80=93) is a serovar of the O:38 (P) serogroup. This serotype was first discovered in the [Belgian Congo](https://pubmed.ncbi.nlm.nih.gov/14387694/) in 1955. FDA found this serovar in [shelled macadamia nut](https://www.ncbi.nlm.nih.gov/datasets/genome/GCA_006427635.1/). NCBI PD link for serovar Korovi: [https://www.ncbi.nlm.nih.gov/pathogens/isolates/#(taxgroup_name:%22Salmonella%20enterica%22)%20AND%20computed_types:(%22serotype=Korovi%22)](https://www.ncbi.nlm.nih.gov/pathogens/isolates/#(taxgroup_name:%22Salmonella%20enterica%22)%20AND%20computed_types:(%22serotype=Korovi%22))

## Genetic Characteristics

 Serovar Korovi is considered rare, and there is no known literature that has studied it.

## Animal Reservoir

 Serovar Korovi has been found in [cattle](https://pubmed.ncbi.nlm.nih.gov/29254708/), [lizards, and snakes](https://pubmed.ncbi.nlm.nih.gov/15074631/).

## Geographical Distribution

 Serovar Korovi has been reported in[the US](https://www.ncbi.nlm.nih.gov/datasets/genome/GCA_006427635.1/), [Ethiopia](https://pubmed.ncbi.nlm.nih.gov/29254708/), and several [European countries](https://atlas.ecdc.europa.eu/public/index.aspx?Dataset=27&HealthTopic=46&Indicator=887175&GeoResolution=2&TimeResolution=Year&StartTime=2006&EndTime=2022&CurrentTime=2016&Distribution=884316&DistributionRepresentation=B&TimeSeries=region&TimeSeriesRepresentation=T)such as Sweden, Germany, Spain, and the UK.

## Human/Animal Outbreaks

 There have been no recent human outbreaks linked to this serovar.

## Border Rejections

 There have been no recent border rejections linked to this serovar.
