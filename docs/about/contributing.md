# Contributing

We welcome contributions from researchers, public health professionals, and food safety experts.

## How to Contribute

1. **Fork** this repository
2. **Create a branch** for your changes
3. **Add or update** serovar pages using the [template](../serovars/_template.md)
4. **Submit a pull request** with a description of your changes

## Guidelines

- Use the serovar page template for consistency
- Cite all sources in the **Relevant Links** section
- For outbreaks, include year, location, source, and case count
- Use archived links (via [Wayback Machine](https://web.archive.org/)) for any expired URLs
