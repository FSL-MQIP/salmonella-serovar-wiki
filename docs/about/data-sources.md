# Data Sources

The Salmonella Serovar Wiki consolidates information from publicly available data sources, including peer-reviewed scientific literature, surveillance reports, genomic databases, and regulatory agency publications.

## Primary Sources

| Source | Description |
|--------|-------------|
| [CDC FoodNet](https://www.cdc.gov/foodnet/) | U.S. foodborne illness surveillance data |
| [CDC NORS](https://wwwn.cdc.gov/norsdashboard/) | National Outbreak Reporting System |
| [NCBI Pathogen Detection](https://www.ncbi.nlm.nih.gov/pathogens/) | Genomic surveillance and isolate data |
| [FDA CORE Network](https://www.fda.gov/food/foodborne-pathogens/coordinated-outbreak-response-and-evaluation-core-network) | U.S. outbreak investigations |
| [ECDC](https://www.ecdc.europa.eu/) | European surveillance and outbreak reports |
| [RASFF](https://food.ec.europa.eu/safety/rasff-food-and-feed-safety-alerts_en) | EU Rapid Alert System for Food and Feed |
| [WHO](https://www.who.int/) | Global public health data and reports |
